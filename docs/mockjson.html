在线Mock JSON API | 快速生成模拟数据 - GoTool在线工具集

开发工具指南

此工具提供在线 Mock JSON 工具，具备高性能 JSON 服务，支持 URL 参数自定义。该工具允许用户快速生成和模拟 JSON 数据，方便进行 API 开发和测试。支持根据 URL 参数动态调整响应数据，提升开发效率和测试灵活性.

使用说明

功能介绍

1.将输入的 json 串作为返回参数
2.支持 url 参数
3.生成的 url 主要通过 token 参数来确定输出, 切勿修改 token

使用示例

1.无请求参数

[
	{
		"id": 1,
		"name": "test",
		"age": 10,
		"cls_name": "初一",
		"addr": "四川成都"
	},
	{
		"id": 2,
		"name": "test",
		"age": 10,
		"cls_name": "初二",
		"addr": "四川成都"
	}
]

处理后

http://localhost:9000/toolapi/mockjson?come_from=2&token=d491e4e9771c051021fdcf568cc369c4

2.有参数

{
	"id": 1,
	"name": "test",
	"age": 10,
	"cls_name": "初一",
	"addr": "四川成都"
}

处理后

http://localhost:9000/toolapi/mockjson?come_from=2&user_id=1&token=c7e6d7031b9911793b0b5ae83f50a9a8
