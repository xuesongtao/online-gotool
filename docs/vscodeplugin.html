GoTool VsCode插件 | SQL/JSON/XML转换工具 | GoTool在线工具集

开发工具指南

GoTool VsCode插件提供SQL、JSON、XML等多种数据格式转换功能，支持实时预览和一键转换，是开发者必备的编码助手。

使用说明

GoTool VsCode Plugin

  • 将 GoTool 的强大功能直接集成到 VSCode 编辑器中

使用说明

  1. 在编辑器中选中需要转换的文本（SQL 或 JSON）。
  2. 点击右键，选择 GoTool: SQL Convert Tools 或 GoTool: JSON Convert Tools 等。
  3. 在子菜单中选择目标格式。
  4. 转换结果将在预览窗口展示，确认无误后可点击按钮一键替换原选中文本。

插件配置

在 VSCode 中打开设置（快捷键 Cmd + , 或 Ctrl + ,），搜索 gotool 即可找到以下专属配置项：

| 配置项 | 说明 | 默认值 | | :-------------- | :-------------------------------------------------- | :--------------------------- | | gotool.webUrl | GoTool 主站服务地址 | https://gotool.top | | gotool.apiUrl | 后端 API 服务地址，如果您有私有化部署环境可修改此项 | https://gotool.top/toolapi | | gotool.token | 用户授权 Token，用于同步云端数据和验证高级功能权限 | （无） |

如何获取 Token？ 请前往 GoTool 网站登录您的账号，进入【个人中心】后点击 【授权 VSCode】 按钮即可一键复制 Token 并在此处配置。
