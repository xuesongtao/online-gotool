在线实时传输 (TransferTxt)

在线实时传输是一个基于 Web 的文本同步工具，支持多端实时互传文本内容。

核心特性
  • 实时同步：基于 SSE (Server-Sent Events) 和 WebSocket 技术实现，优先使用 SSE 以获得更好的连接稳定性, 同时3个设备同步。
  • 零延迟：一旦一方修改，所有连接同一通道的终端将立即收到更新。
  • 多端支持：支持 PC、手机等多种设备浏览器同时访问.
存储规则
  • 有效期：文本内容将在服务器中保留 10 分钟。
  • 计算方式：有效时间以最后一次修改开始计算。每次更新文本内容，倒计时将重新开始。
  • 自动清理：超过有效期未更新，内容将被系统自动清除，确保隐私安全。
使用场景
  • 跨设备粘贴：在电脑上复制一段长文本或链接，手机端直接打开即可获取。
  • 临时备忘：快速记录一些临时信息，多点同步。
  • 简单协作：与伙伴实时共享并编辑同一段文本。
  • 服务器日志：通过 curl 用于发送服务器日志信息。
安全与隐私
  • 传输过程加密，不进行持久化存储。
  • 内容到期自动销毁，不留痕迹。
