开发工具指南

git 常用命令在线速查表。轻松查找和掌握 Git 初始化、提交、分支管理、合并、回滚等高频命令，提升代码版本控制效率。适合开发者、程序员和 DevOps 专业人士，随时随地使用，助你高效处理 Git 操作。

使用说明

将本地项目推送到远程仓库

git init 

git add .

git commit -m "feat: xxx"

git remote add origin https://xxx

git pull --rebase origin xxx

git push -u origin xxx

编写了 .gitignore 后立即生效

# 清除缓存  
git rm -r --cached .  

# 重新trace file  
git add . 

# 提交和注释 
git commit -m "update .gitignore"

打 tag

# 打 tag
git tag -a v1.0.0 -m "feat:xxx"

# 显示内容
git show v1.0 

# 将此 tag 推送到远程
git push origin v1.0.0 

# 如果要一次推送所有本地新增的标签上去
git push origin --tags

删除操作

# 删除本地 tag
git tag -d tagName 

# 删除远程 tag
git push origin :refs/tags/tagName

# 删除本地分支
git branch -d name
# 强制删除
git branch -D name

# 删除远程分支
git push origin -d name
