Core Description

GO在线工具集提供多功能在线工具，涵盖SQL转换与格式化、编码与加密、图片处理、数据转换与解析、正则表达式测试、IP与时间查询、以及文本相似度计算。支持多数据库SQL转换为Go struct、XORM、GORM、Proto等代码，并能进行URL、Base64编码/解码、MD5加密、图片裁剪压缩、YAML/JSON/XML格式转换，以及正则表达式替换匹配和生成代码。工具还支持精准IP查询、时间与时间戳转换，以及使用Levenshtein和Jaccard算法计算文本相似度，为开发者提供全面、高效的解决方案。
