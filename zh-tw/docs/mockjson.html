在線mock json api|快速生成模擬數據 - gotool在線工具集

开发工具指南

此工具提供在線 mock json 工具，具備高性能 json 服務，支持 url 參數自定義。該工具允許用戶快速生成和模擬 json 數據，方便進行 api 開發和測試。支持根據 url 參數動態調整響應數據，提升開發效率和測試靈活性.

使用说明

功能介紹

1.將輸入的 json 串作為返回參數
2.支持 url 參數
3.生成的 url 主要通過 token 參數來確定輸出, 切勿修改 token

使用示例

1.無請求參數

[
    {
        "id": 1,
        "name": "test",
        "age": 10,
        "cls_name": "初一",
        "addr": "四川成都"
    },
    {
        "id": 2,
        "name": "test",
        "age": 10,
        "cls_name": "初二",
        "addr": "四川成都"
    }
]

處理後

http://localhost:9000/toolapi/mockjson?come_from=2&token=d491e4e9771c051021fdcf568cc369c4

2.有參數

{
    "id": 1,
    "name": "test",
    "age": 10,
    "cls_name": "初一",
    "addr": "四川成都"
}

處理後

http://localhost:9000/toolapi/mockjson?come_from=2&user_id=1&token=c7e6d7031b9911793b0b5ae83f50a9a8
