GoTool VsCode plugin| SQL/JSON/XML conversion tool| GoTool online toolset

开发工具指南

The GoTool VsCode plug-in provides multiple data format conversion functions such as SQL, JSON, and XML. It supports real-time preview and one-click conversion. It is a must-have coding assistant for developers.

使用说明

GoTool VSCode Plugin

  • Integrate the common functions of GoTool into the VSCode editor

Usage Instructions

  1. Select the text you want to convert in the editor (SQL or JSON).
  2. Right-click and choose GoTool: SQL Convert Tools or GoTool: JSON Convert Tools, etc.
  3. Select the target format from the submenu.
  4. The conversion result will be displayed in a preview window. Once confirmed, you can click the button to replace the original selected text with one click.

Plugin Configuration

Open VSCode Settings (shortcut Cmd + , or Ctrl + ,) and search for gotool to find the following configuration options:

| Option | Description | Default Value | | :-------------- | :----------------------------------------------------------------------------- | :--------------------------- | | gotool.webUrl | GoTool main website service URL | https://gotool.top | | gotool.apiUrl | Backend API service URL; can be modified for private deployment environments | https://gotool.top/toolapi | | gotool.token | User authorization token for syncing cloud data and accessing premium features | (none) |

How to get a Token? Go to the GoTool website, log into your account, enter the Profile page, and click the “Authorize VSCode” button to copy your Token and paste it here.
